ADM-3A

Bill Joy created vi using a ADM-3A terminal. Its keyboard has distinct features that heavily influenced vi design.

The ESC was located in place of the actual TAB key. Below, the CTRL key was placed instead of the CAPS-LOCK key.

Each h, j, k and l key was labelled with the arrows we know and love.

Finally, on the top right corner, the Home key was also the ~ key.
